# GitHub User Activity

Projeto desenvolvido em Python que consome a API pública do GitHub para exibir as atividades recentes de um usuário.

## Funcionalidades

- Consulta de atividades recentes
- Tradução dos tipos de eventos
- Exibição da data formatada
- Limitação para os 10 eventos mais recentes
- Tratamento de erros

## Tecnologias

- Python
- Requests
- GitHub REST API

## Como executar

```bash
pip install -r requirements.txt
python main.py
